== Description ==
Animated number counter for your website.
You just need to install and activate the plugin.
Select the required settings in the admin panel, and set the DP Number Counter template for the required page.
Enjoy!

== Installation ==
1. Install the plugin from the plug-in store.
2. Activate the plugin
3. Configure the plugin (Settings -> Animated Number Counter)
4. Set page template
